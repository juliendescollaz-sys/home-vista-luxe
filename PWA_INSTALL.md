# Installation PWA - Neolia SmartHome

## Mode Plein Écran (Standalone)

Cette application est une PWA (Progressive Web App) qui peut être installée sur votre écran d'accueil et s'ouvrir en mode plein écran, sans les barres du navigateur.

## Installation sur iOS (iPhone/iPad)

1. Ouvrez l'application dans **Safari** (obligatoire sur iOS)
2. Appuyez sur le bouton **Partager** (icône carré avec flèche vers le haut)
3. Faites défiler vers le bas et appuyez sur **"Ajouter à l'écran d'accueil"**
4. Confirmez en appuyant sur **"Ajouter"**
5. Une icône Neolia apparaîtra sur votre écran d'accueil

### Lancement en mode plein écran

- Lancez l'app depuis l'**icône sur l'écran d'accueil** → Mode plein écran (pas de barres Safari)
- Ouvrez depuis Safari ou un favori → Mode navigateur normal (barres visibles)

## Installation sur Android

1. Ouvrez l'application dans **Chrome**
2. Appuyez sur le menu (⋮) en haut à droite
3. Sélectionnez **"Installer l'application"** ou **"Ajouter à l'écran d'accueil"**
4. Confirmez l'installation
5. L'icône Neolia apparaîtra sur votre écran d'accueil

## Vérification

Après installation, l'application :
- ✅ S'ouvre en mode plein écran (sans barres du navigateur)
- ✅ Fonctionne hors ligne (cache minimal)
- ✅ Respecte les zones sûres (encoche iPhone)
- ✅ Affiche le splash screen au démarrage

## Support

L'application détecte automatiquement si elle est lancée en mode PWA via le paramètre `?source=pwa`.
